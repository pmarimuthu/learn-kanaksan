# Physical Quantities & SI Units

<ConceptMeta chapter="1" section="1.2" page="1–5" />

::: idea
To **measure** something is to compare it with an agreed standard called a **unit**. Saying "the rope is 5" is meaningless; "5 metres" is not.

The world agrees on one system — the **SI** (Système International). It has just **seven base units**, and every other unit (for speed, force, energy…) is a **derived unit** built by combining them.
:::

<ConceptHero type="si-units" />

::: formula
A few SI base quantities and their units:

- length → metre (m)
- mass → kilogram (kg)
- time → second (s)

**Derived units** combine base units, for example:

$$[\text{speed}] = \text{m s}^{-1} \qquad 1\,\text{N} = 1\,\text{kg}\cdot\text{m}\cdot\text{s}^{-2}$$

Symbols: m = metre, kg = kilogram, s = second, N = newton (the unit of force).
:::

::: real
A carpenter in Bengaluru and a steel mill in Germany can fit parts together only because both use the **same metre and the same kilogram**. GPS, online trade, medicine doses and space missions all depend on everyone sharing one set of standards.
:::

::: try
Measure your height with a piece of thread, then lay the thread against a ruler marked in centimetres and convert to metres. Notice you *needed* the ruler — a standard — before the number meant anything.
:::

::: whatif
Long ago a "foot" was literally the length of the local king's foot, so it changed from place to place. How would science or trade work if your metre and mine were different lengths?
:::

::: notes
- **7 SI base units:** metre (m), kilogram (kg), second (s), ampere (A), kelvin (K), mole (mol), candela (cd).
- **Derived units** are combinations, e.g. speed m s⁻¹, force N = kg m s⁻².
- Unit symbols are not pluralised and follow case rules: `5 kg` (not 5 kgs); the newton's symbol is a capital **N**, the unit name "newton" is lowercase.
- **SI prefixes** scale units: kilo (10³), centi (10⁻²), milli (10⁻³), micro (10⁻⁶), mega (10⁶).
:::
